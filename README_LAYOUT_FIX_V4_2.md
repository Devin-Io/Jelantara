# JELANTARA Layout Refinement V4.2

Perbaikan ini menuntaskan overlap pada bagian **Tentang JELANTARA**.

## Perubahan
- Kartu mengambang “Prinsip utama” dihapus dari luar frame.
- Informasi “Circular economy” dan “Prinsip utama” disatukan dalam satu panel internal.
- Tidak ada elemen yang menabrak batas frame atau kolom kanan.
- Orbit utama dinaikkan agar memiliki ruang aman dari panel informasi bawah.
- Label orbit disesuaikan untuk desktop, tablet, dan mobile.
- Tinggi visual disesuaikan agar proporsinya stabil pada berbagai viewport.

## Instalasi patch
1. Hentikan Vite dengan `Ctrl + C`.
2. Salin folder `src` dari patch ke project utama.
3. Pilih **Replace the files in the destination**.
4. Jalankan `npm run dev`.
5. Tekan `Ctrl + Shift + R` di browser.

Tidak ada dependency baru.
